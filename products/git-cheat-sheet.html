Git Cheat Sheet

Tenha em mãos um guia de referência rápida para o Git. Este Cheat Sheet foi desenhado para acelerar seu fluxo de trabalho, com os comandos mais utilizados no dia a dia.

Nunca mais perca tempo procurando como fazer um rebase interativo ou como resetar um commit.

O que você encontrará:

  • Comandos de configuração inicial.
  • Operações básicas: add, commit, push, pull.
  • Trabalho com branches: criar, trocar, listar e fundir.
  • Comandos avançados como rebase, cherry-pick e stash.
  • Dicas para resolver conflitos e visualizar o histórico.
