Docker Cheat Sheet

Acelere seu desenvolvimento com containers. Este guia de referência rápida para Docker coloca os comandos mais importantes ao seu alcance.

Desde a criação de imagens até o gerenciamento de volumes e redes, encontre o que você precisa sem esforço.

O que você encontrará:

  • Comandos para ciclo de vida de containers (run, stop, start, rm).
  • Construção e gerenciamento de imagens (build, push, pull, rmi).
  • Trabalho com Docker Compose para múltiplos containers.
  • Gerenciamento de volumes e redes.
  • Dicas para otimizar seus Dockerfiles.
