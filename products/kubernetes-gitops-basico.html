Kubernetes: GitOps Básico

Neste e-book, você vai mergulhar no mundo do GitOps, aprendendo a utilizar o Argo CD para automatizar a entrega de aplicações em um cluster Kubernetes rodando no Docker Desktop.

É o guia perfeito para quem quer modernizar seus processos de deploy, garantindo mais segurança, rastreabilidade e eficiência.

O que você vai aprender:

  • Os conceitos fundamentais de GitOps.
  • Como instalar e configurar o Argo CD.
  • Criar e gerenciar aplicações de forma declarativa.
  • Sincronizar o estado do seu cluster com um repositório Git.
  • Boas práticas para um fluxo de trabalho de entrega contínua.
